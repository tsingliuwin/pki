# PKI: 个人知识内化系统

**基于极小模型的本地化“第二大脑”实现方案。**

PKI 是一个高性能系统，旨在将你的非结构化文档转化为极小模型（Qwen3-0.6B）的内在权重。它专注于 **“权重演化”** 而非传统的“Prompt 注入 (RAG)”。

## 🚀 PKI 的哲学
> “一次性内化成本，永久性本地收益。”

PKI 将文档蒸馏为高质量的 QA 对，并通过 LoRA 微调将这些知识“刻录”进模型。最终产出一个能在普通 CPU 上流畅运行、真正“懂你数据”的 GGUF 模型文件。

## 🏗 工作流：职责分离架构

### 1. 数据摄入 (`pki-ingest` / 纯 Rust) - [100% 可交付]
- **功能**：语义分块 + LLM 驱动的 QA 数据集自动生成。
- **硬件**：普通 CPU 即可。

### 2. 知识内化 (`pki-trainer` / Python) - [一次性离线步骤]
- **功能**：编排 PEFT/Unsloth 进行 LoRA 微调。
- **起点**：BF16 原始权重 (1.15GB Safetensors)。
- **硬件**：**NVIDIA GPU (8GB+ 显存)** 或 **Apple M 系列芯片**。
- **产出**：合并权重并量化为 GGUF。

### 3. 本地推理 (`pki-engine` / 纯 Rust) - [100% 可交付]
- **功能**：基于 `candle` 的极速量化推理。
- **硬件**：任何支持 AVX2 的 CPU。
- **性能**：内存占用 < 800MB，冷启动 < 200ms。
